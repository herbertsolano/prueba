Definición y uso


El reverse()método invierte el orden de los elementos en una matriz.

Nota: este método cambiará la matriz original.


Ejemplos: Invierta el orden de los elementos en una matriz:


var fruits = ["Banana", "Orange", "Apple", "Mango"];

fruits.reverse();


Detalles técnicos


Valor de retorno: Una matriz, que representa la matriz después de que se ha invertido.

Versión de JavaScript: ECMAScript 1.
